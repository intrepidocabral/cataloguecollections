# cinecatalog

# Requirements
* python 3.5
* mysql

## Deploy

Crie o env:

```
$ virtualenv env
```

Ative o env:
```
$ source env/bin/activate
```

* Crie a base:
```
$ python3.5 manage.py migrate
```

## Atalhos

### Rodando a aplicação localmente
1. Ative o env
2. Rode o runserver
```
$ python3.5 manage.py runserver
```
